My first multiple page web design project. This site helped me become more comfortable with using external CSS. I learned new CSS attributes such as opacity, hover, and transition effects. MOVE

This landing page was created on pure curiosity. I wanted to learn how parallax sites behaved at its simplest form. To my surprise, the creation of a parallax sites was simpler than I originally imagine it to be. With some simple DIV containers and fixed background images I was able to accomplish the desired effect.
